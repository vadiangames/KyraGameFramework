ABSTRACT RENDERING DEVICE

Description

The AbstractRenderDevice (short: ARDAPI) abstracts the rendering process.
Implementations must implement the abstract classes of the API.
At the same time, the ARDAPI also serves as an interface for projects that uses the rendering devices.

Experimental render device implementations

  • OpenGL 3.3

Planned render device implementations:

  • OpenGL 2.1
  • OpenGL 4.2
  • OpenGL 4.6
  • Vulkan
  • DirectX 11
  • DirectX 12

Usage:

To create a renderer you need to have a window-instance.

#include <KyraGameFramework/Window.hpp>

kyra::WindowSettings windowSettings;
 windowSettings.width = 800;
 windowSettings.height = 600;
 windowSettings.title = L"Example";

kyra::Window window;
 if(window.create(windowSettings)) {
  while(window.isOpen()) {
   window.processEvents();
 }
}

Have a look at the window documentation for detailed informations about creating a window

If you succesfully created a window, you can now create the renderer instance.
